# SWAMP Java API

A Java API for https://www.mir-swamp.org/.

## Prerequisites

* Java 8+
* [Maven](https://maven.apache.org/) 3.0+

## Building

`mvn clean install`

## Generating Javadoc

`mvn javadoc:javadoc`

Then point your browser at target/site/apidocs/index.html.
